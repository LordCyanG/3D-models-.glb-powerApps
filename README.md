# CyanG

Este es un repositorio donde se subierno distintos modelos .glb para su uso en una aplicación de Microsoft Power Apps.
Son libres de usarlos PERO RECUERDEN, estos modelos no los cree yo, son descargados de la página "https://www.cgtrader.com/" en la seccion "3D Models" y los filtros de "FREE" y "GLTF (.Gltf .Glb)".
Gracias a los creadores de estos modelos.

Si por alguna razon encuentran este repositorio y lo utilizan para la misma plataforma utilicen este formato de enlace: "https://raw.githubusercontent.com/LordCyanG/CyanG/main/'Nombre del archivo'.glb"

En una página de Microsoft explicaban como hacerlo, aquí lo que me sirvio:

Para usar GitHub
-Asegúrese de que su repositorio de Git esté establecido en Público.
-Desplácese hasta el archivo. Por ejemplo, https://github.com/microsoft/experimental-pcf-control-assets/blob/master/robot_arm.glb.
-Elimine /blob/.
-Reemplace https://github.com con https://raw.githubusercontent.com. Ahora tiene acceso sin formato a su archivo. Por ejemplo, https://raw.githubusercontent.com/microsoft/experimental-pcf-control-assets/master/robot_arm.glb.

Muchas gracias.
